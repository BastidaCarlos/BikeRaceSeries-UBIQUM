# Bike Race Series

Proyecto del módulo 1 - Desarrollo Web Fullstack.
Sitio web, construido con HTML y CSS.

## Páginas

| Archivo        | Descripción      |
| -------------- | ---------------- |
| `index.html`   | Página de inicio |
| `about.html`   | Sobre nosotros   |
| `contact.html` | Contacto         |

## Estructura de estilos

El proyecto incluye 3 diseños. Para cambiar el estilo activo, edita la línea
`<link>` en cada archivo HTML:

```html
<!-- Activar estilo 1 -->
<link rel="stylesheet" href="styles/brs_1.css" />

<!-- Activar estilo 2 -->
<link rel="stylesheet" href="styles/brs_2.css" />

<!-- Activar estilo 1 -->
<link rel="stylesheet" href="styles/brs_3.css" />
```

El estilo activo por defecto es **brs_1.css**

## Estructura de carpetas

```
/
├── assets/

├── styles/
│   ├── brs_1.css   ← propuesta 1
│   ├── brs_2.css   ← propuesta 2
│   └── brs_3.css   ← propuesta propia
├── index.html
├── about.html
├── contact.html

```
